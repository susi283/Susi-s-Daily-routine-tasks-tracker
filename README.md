Mission Control (Home)
Interactive Tasking: Add, edit, and toggle tasks with dynamic neon feedback.

LVL_PROGRESS HUD: Real-time visual tracking of your daily completion percentage.

HUD Summary: Instant visibility into Total, Completed, and Remaining objectives.

Hall of Fame (Archive)
Categorized Success: View completed missions organized by category (Neural Upgrades, Vitality, etc.).
Protocol Restore: Accidentally finished a quest? Use the restore function to bring it back to active status.
Relaxation Protocol (Cheat Day)
Guilt-Free Leisure: Log your movies, gaming hours, and screen time.
Reward System: Tracks how many "Crystals" (non-learning tasks) you've earned to justify your downtime.
Intelligence Stream
AI Motivation: Daily quotes generated by AI to keep you firing.
Tech Stack
Frontend: React 19 (ES6+ Modules)
Styling: Tailwind CSS with custom Neon-Glow configurations
AI Engine: Gemini
State & Persistence: React Hooks + Browser LocalStorage
Installation & Setup
Clone the Repository:

git clone https://github.com/susi283/susinds-daily-tasks.git
cd susinds-daily-tasks
Install Dependencies:

npm install
Boot the System:

npm start
